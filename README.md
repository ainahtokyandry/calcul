Cette application est codée avec php en backend et HTML, CSS et JavaScript en frontend. Elle a été conçue pour, pour le moment, effectuer des calculs simples avec les signes "+", "-", "*", "/".
Ouvrez le fichier /public/index.html dans un navigateur qui active JavaScript et vous pourrez commencer à l'utiliser.

L'application a été créée suite à un test que j'ai fait dans une société. Je n'avais pas assez de temps pour le finir. Quand j'étais arrivé chez moi, j'ai décidé de le refaire. Et voilà le résultat.
